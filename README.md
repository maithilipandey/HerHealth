# 🌸 HerHealth

A women's wellness web application designed to help women track their health, understand their bodies, and focus on self-care in a safe digital space.

---

## 💖 Live Website

🌐 https://her-health-teal.vercel.app/

---

## ✨ Features

🌸 Period Tracker  
💕 Mood Tracker  
🧘 Wellness Tips  
🔒 Private & Secure Health Tracking  
📊 Clean and calming user interface  

---

## 🛠 Tech Stack

- **Frontend:** Next.js, React
- **Styling:** Tailwind CSS
- **Backend (planned):** Node.js / Express
- **Database (planned):** MongoDB

---

## 🎨 UI Design

The interface uses soft pastel colors and smooth animations to create a calm, welcoming experience focused on women's wellness.

---

## 🚀 Run Locally

Clone the repository

```bash
git clone https://github.com/maithilipandey/HerHealth.git
```

Go to project folder

```bash
cd HerHealth
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open in browser

```
http://localhost:3000
```
